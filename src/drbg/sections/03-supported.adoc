
[#supported]
== Supported DRBGs

The following deterministic random bit generator algorithms *MAY* be advertised by the ACVP compliant cryptographic module. The list is in the form "algorithm / revision."

* hashDRBG / 1.0
* hashDRBG / SP800-90Ar1
* hmacDRBG / 1.0
* hmacDRBG / SP800-90Ar1
* ctrDRBG / 1.0
* ctrDRBG / SP800-90Ar1

=== Counter DRBG Triple-DES

The "ctrDRBG-TDES" mode shall only be used with the three-key option of the Triple-DES algorithm.
