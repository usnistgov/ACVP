== Supported DRBGs

The following deterministic random bit generator algorithms *MAY* be advertised by the ACVP compliant cryptographic module. The list is in the form "algorithm / revision."

* hashDRBG / 1.0
* hashDRBG / SP800-90Ar1
* hmacDRBG / 1.0
* hmacDRBG / SP800-90Ar1
* ctrDRBG / 1.0
* ctrDRBG / SP800-90Ar1

=== Counter DRBG Triple-DES

NOTE: TDES is no longer approved as of 1/1/2024; please refer to https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.800-131Ar2.pdf for more information.
